git checkout lv2.0

Traer los últimos cambios del repositorio remoto (por si hiciste push recientemente):
git fetch origin

Hacer merge desde tu rama v0.3:
git merge origin/v0.3


Resolver cualquier conflicto (si los hay), luego:

git add .
git commit -m 'actualizacion ...'
